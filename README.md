## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Install JSON-SERVER
```sh
npm install -g json-server
```

### Watch JSON
```sh
json-server -w ./data/db.json
```
